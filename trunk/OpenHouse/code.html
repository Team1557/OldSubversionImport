Programming Team

The programming team handles the movement and control of the robot.

Programming Language

We have used Java for several competitions. Java allows us to make our code modular (so it's easy to add new features) and efficient (so the controls are responsive)

The Tech

The programming team also plays a part in the tech on the robot, and the control board.

Joysticks

Our control system is made up of three joysticks, two of which are used to drive the robot in a "tank drive" system, the other is used for controlling the arms and other scoring mechanisms.
